## 📖 About The Project

Lumina Creative is a lightweight, frontend-only project designed to showcase modern web design principles without relying on heavy frameworks. It demonstrates a strong grasp of core web technologies, fluid layouts, and mobile-first design thinking.

### ✨ Key Features
* **Fully Responsive:** Extensive use of CSS media queries to ensure a seamless layout across desktops, tablets, and mobile devices.
* **Modern UI/UX:** Clean, aesthetically pleasing interface tailored for a creative agency.
* **Vanilla JavaScript:** Interactive elements (like mobile navigation toggles) implemented using lightweight, pure JS.
* **Zero Dependencies:** Built from scratch without Bootstrap or Tailwind, showcasing raw CSS proficiency.

## 🛠️ Tech Stack

* **Markup:** HTML5 (Semantic Structure)
* **Styling:** CSS3 (Flexbox/Grid, Custom Variables, Media Queries)
* **Interactivity:** Vanilla JavaScript (ES6+)
